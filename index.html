Spark配置参数详解

Spark提供三个位置用来配置系统：

  • Spark属性：控制大部分的应用程序参数，可以用SparkConf对象或者Java系统属性设置
  • 环境变量：可以通过每个节点的conf/spark-env.sh脚本设置。例如IP地址、端口等信息
  • 日志配置：可以通过log4j.properties配置

spark参数配置说明

发表于 2019-10-18 分类于 python ， spark

spark配置修改方式

Spark有以下三种方式修改配置：

  1. Spark properties （Spark属性）

    可以控制绝大多数应用程序参数，而且既可以通过 SparkConf 对象来设置，也可以通过Java系统属性来设置。

  2. Environment variables （环境变量）

    可以指定一些各个机器相关的设置，如IP地址，其设置方法是写在每台机器上的conf/spark-env.sh中。

  3. Logging （日志）

    可以通过log4j.properties配置日志。
